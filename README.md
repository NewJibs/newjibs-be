## 🏠 Introduction

### main-service

- <U>**부동산 모의 투자 서비스**</U>
- 예산 100억을 기반으로 부동산에 투자하여 2년 뒤 최대 차익을 내는 것을 목표로 한다.
- 투자 이후에는 차익과 비례한 만큼의 포인트가 적립된다.

### sub-service

- 랭킹 시스템
- 공지사항
- 최신 부동산 뉴스 기사 조회 서비스
